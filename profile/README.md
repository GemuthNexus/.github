## üß¨ Core Philosophy

We believe that managing your own server should feel as **natural** as using a desktop ‚Äî without giving up control to third-party platforms. GemuthNexus is built on the principle that the **Linux kernel** already provides everything you need for security, identity, and isolation. The platform's job is to make that power **accessible** through a modern interface ‚Äî not to replace it.

### üõ°Ô∏è The Pillars

*   **Native Identity** ‚Äî Your web session is your Linux session. Users authenticate with real system credentials, and every permission is enforced by the kernel.
*   **Process Isolation** ‚Äî Every application runs as a dedicated process under the user's own UID. No shared runtimes, no containers, no trust boundaries to configure.
*   **Lean & Efficient** ‚Äî The entire platform idles at ~30 MB RAM. Plugins are spawned on demand and released when idle. No background bloat, no wasted resources.
*   **Extensibility** ‚Äî The platform ships with zero built-in apps. Everything is a plugin ‚Äî installed, updated, and removed independently.
